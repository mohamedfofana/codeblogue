Jeu du morpion

Pré-requis

Il y a neuf positions possibles. Chaque joueur dispose de trois pions de même couleur. Pour poser un pion, il faut cliquer dessus et cliquer sur une case vide. Pour changer un pion de position, cliquer sur le pion et cliquer sur la position. On peut déplacer un pion sur toutes les lignes qui partent de cette position.


Principe :

Le jeu se joue avec deux joueurs. Chacun choisit une couleur ou une forme.
Le premier joueur pose son pion et le second joueur pose le sien. Et ainsi de suite jusqu'à ce que chacun ait posé ses trois pions.
Si aucun joueur n'a réussi à aligner ces trois pions sur une même ligne droite, il peut essayer de les déplacer sur la grille afin d'obtenir l'alignement.
Le joueur qui pose ces trois pions sur une ligne droite à gagner. La partie est terminée si aucun joueur ne parvient à aligner ses pions.

Environnement
Environnement technique : Microsoft Visual Studio 2010, Microsoft Visual C# 2010 Express, Microsoft XNA Game Studio 4.0
Langage : Flex
Mode : 2 joueur
